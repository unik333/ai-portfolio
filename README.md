# 🤖 unikAI — AI Engineer Portfolio

A stunning, dark-themed portfolio website showcasing AI engineering skills, projects, and capabilities.

**🌐 Live Site:** https://unik333.github.io/ai-portfolio/

## ✨ Features
- Dark navy + purple premium design
- Live GitHub project cards fetched via API
- Typing animation with AI specialties
- Animated orbit rings, floating blobs, skill bars
- Contact form, services section, stats counter
- Fully responsive for all devices

## 🛠 Built With
- Vanilla HTML5, CSS3, JavaScript
- GitHub API for live project data
- Font Awesome icons + Google Fonts (Inter, Space Grotesk)
- GitHub Pages for deployment

## 📂 Structure
```
├── index.html      # Main page
├── style.css       # All styles & animations
├── script.js       # GitHub API, typing, interactions
└── netlify.toml    # Netlify config (optional)
```
